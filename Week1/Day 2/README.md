## Day 2 - Python Fundamentals

### Overview

Covered core Python fundamentals needed for data science work: data types, control flow, functions, list comprehensions, and object-oriented programming. Work is split across two notebooks.

### Files

- `day2_python_fundamentals.ipynb` - detailed notes and worked examples for every topic
- `Hands_On_Lab.ipynb` - the official required exercises

### Topics Covered

- Core data types: `int`, `float`, `str`, `bool`, `list`, `tuple`, `dict`, `set`, `None`
- Control flow: `if`/`elif`/`else`, `while` loops, `for` loops, `break`/`continue`
- Functions: parameters, default arguments, return values, docstrings
- List comprehensions (including filtering with conditions)
- Object-Oriented Programming:
  - Basics: classes, `__init__`, `self`, attributes, methods, class vs. instance variables
  - Advanced (bonus): inheritance, polymorphism, encapsulation, abstraction - explored through ML-relevant examples (model classes) to prepare for Scikit-learn/PyTorch later in the program

### Hands-On Lab

- Function that takes a list of numbers and returns the mean, min, and max as a dictionary
- Rewrote a `for`-loop that filters even numbers as a single list comprehension
- Built a small `DataRecord` class with two attributes and a `display()` method
